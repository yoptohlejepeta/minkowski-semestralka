# Minkowského suma a její uplatnění

Tato práce byla vypracována v rámci předmětu KI/PDSA. Práce je implementována v
jazyce Julia za pomocí knihovny [Pluto.jl](https://plutojl.org/). Statický
.html export je dostupný na adrese
[yoptohlejepeta.github.io/minkowski-semestralka/](https://yoptohlejepeta.github.io/minkowski-semestralka/).

Pro lokální spuštění je potřeba:

- Julia (^1.12.4)
- balík Pluto.jl (v0.20.21)

Spuštění pomocí příkazu:

```julia
using Pluto; Pluto.run()
```
